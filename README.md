## 📖 Proje Hakkında

Bu platform, **UI/UX Designer'ların** HTML ve CSS dillerini sıfırdan öğrenip gerçek dünya projeleri üretebilmesi için hazırlanmış kapsamlı bir eğitim kaynağıdır. Tasarım bilgisi olan ancak kod tarafına adım atmak isteyen herkes için idealdir.

Tüm içerik **Türkçe** olarak hazırlanmış olup; interaktif ders sayfaları, canlı kod örnekleri, hızlı başvuru cheatsheet'leri ve kolaydan zora sıralanan 10 gerçek dünya projesi içermektedir.

## ⚡ Başlarken

Projeyi yerel ortamınızda çalıştırmak son derece basittir; herhangi bir kurulum veya derleme adımı gerekmez.

### 1. Repoyu Klonlayın

```bash
git clone https://github.com/waroi/html-css-for-designers.git
cd html-css-for-designers
```

### 2. Tarayıcıda Açın

Ana sayfayı tarayıcınızda açmak için `index.html` dosyasını çift tıklayın **ya da** bir canlı sunucu kullanın:

```bash
# VS Code Live Server eklentisi ile (önerilen)
# → Dosyaya sağ tıklayın → "Open with Live Server"

# Python ile basit HTTP sunucusu
python -m http.server 3000
# Ardından tarayıcıda: http://localhost:3000
```

> **💡 İpucu:** VS Code kullanıyorsanız [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) eklentisini yüklemeniz, dosyaları kaydettikçe tarayıcının otomatik yenilenmesini sağlar.

### 3. Nereden Başlamalı?

```
index.html  →  mufredat/index.html  →  mufredat/dersler/ders01-html-temelleri.html
```

1. **`index.html`** — Ana sayfayı açın, platformun genel yapısına göz atın.
2. **Müfredat** — `mufredat/index.html` ile derslerin kapsamını inceleyin.
3. **Dersler** — `mufredat/dersler/` klasöründen `ders01` ile başlayıp sırayla ilerleyin.
4. **Projeler** — Her 2–3 dersin ardından ilgili projeleri yapın.
5. **Cheatsheet** — Bir etiketi ya da CSS özelliğini unuttuğunuzda cheatsheet'e başvurun.

### 🧠 Ders Çalışma Stratejisi

> **"Oku → Dene → Uygula"** döngüsünü takip edin.

1. **Önce dersi okuyun** — Her dersi baştan sona geçin, kod örneklerini inceleyin.
2. **Canlı deneyin** — Derslerdeki kod bloklarını kendi editörünüze kopyalayıp değiştirerek ne olduğunu gözlemleyin.
3. **Not alın** — Karmaşık geldiğini düşündüğünüz konuları not edin; o notlar cheatsheet'e başvurmanız için faydalıdır.
4. **Projeyi yapın** — O derse ait projeyi sıfırdan kodlamaya çalışın, kaynak koduna sadece takıldığınızda bakın.
5. **Üzerine koyun** — Projeyi bitirince kendi fikirlerinizle geliştirin (renk, font, yeni bölüm eklemek gibi).

### ⚠️ Sık Yapılan Hatalar

- **Projeyi "sonra yaparım" bırakmak** — Her dersin ardından en az bir kısa alıştırma proje yapın.
- **Kodu anlamadan kopyalamak** — Her satırın ne işe yaradığını anlamaya çalışın.
- **Yalnızca okumak** — HTML ve CSS, **yazarak** öğrenilir. Klavyenizi kullanın!
- **Hataları görmezden gelmek** — Tarayıcı DevTools konsolunu açık tutun, uyarıları inceleyin.
- **7. Dersi Atlamak** — BEM, CSS değişkenleri ve erişilebilirlik, profesyonel kod yazmanın temelidir.
