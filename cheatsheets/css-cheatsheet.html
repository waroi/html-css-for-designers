1️⃣ CSS Bağlama Yöntemleri

CSS kodunuzu HTML sayfanıza entegre etmenin 4 farklı yolu

Inline Stil
HTML etiketine doğrudan yazılan CSS. Hızlı testler için iyidir, fakat profesyonel projelerde önerilmez.
Örnek:
<p style="color: red; font-size: 18px;">Kırmızı metin</p>
Kullanım: Hızlı testler, özel durumlar
<style> Etiketi (Internal)
HTML dosyasının <head> kısmına yazılan CSS. Tek sayfalık siteler için uygun.
Örnek:
<head>
    <style>
        p { color: blue; }
        .highlight { background: yellow; }
    </style>
</head>
Dış CSS Dosyası (External)
Ayrı bir .css dosyasında yazılan CSS. En profesyonel ve tavsiye edilen yöntem.
HTML'de:
<link rel="stylesheet" href="styles.css">
styles.css dosyasında:
p { color: green; }
.container { width: 100%; }
✅ Avantajları: Yeniden kullanılabilir, dosya boyutu küçük, bakım kolay
@import Kuralı
Bir CSS dosyasından başka CSS dosyasını içeri aktarır. Dış dosyaların başında kullanılır.
Örnek:
@import url('reset.css');
@import url('typography.css');

body { font-family: Arial; }
⚠️ Not: Dosya sayısı arttıkça sayfa yükleme hızı etkilenebilir

🎯 Seçiciler

CSS'de hangi HTML öğelerine stil uygulanacağını belirten kurallar

Element Seçici
HTML etiket adıyla direkt seçim. Tüm aynı etiketlere etki eder.
Örnek:
p { color: #333; }
h1 { font-size: 32px; }
button { padding: 10px; }
Class Seçici (.class)
Bir sınıfla işaretlenmiş öğeleri seçer. Nokta (.) ile başlar. Bir sayfada birçok öğeye aynı sınıfı verebilirsiniz.
CSS:
.highlight { background: yellow; }
.btn-primary { background: #6366f1; color: white; }
HTML:
<p class="highlight">Vurgulanmış</p>
<button class="btn-primary">Tıkla</button>
ID Seçici (#id)
Bir ID'yi seçer. Hash (#) ile başlar. Her sayfada sadece bir öğeye kullanılmalıdır.
CSS:
#header { background: linear-gradient(135deg, #6366f1, #10b981); }
#footer { margin-top: 50px; }
HTML:
<div id="header">Logo</div>
Evrensel Seçici (*)
Sayfadaki TÜM öğeleri seçer. Sıfırlama (reset) için kullanılır.
Örnek:
* { margin: 0; padding: 0; box-sizing: border-box; }
Gruplandırma (,)
Birden fazla seçiciyi aynı kuralla seçer. Virgülle ayrılırlar.
Örnek:
h1, h2, h3 { color: #333; font-weight: bold; }
.btn, button { padding: 10px 20px; border: none; }
Soyundan Gelen Seçici (boşluk)
Bir öğenin içindeki tüm alt öğeleri seçer (kaç katman derinse dersin).
Örnek:
.container p { color: blue; }
nav a { text-decoration: none; }
Seçer:
<div class="container">
    <p>Bu seçilir</p>
    <div>
        <p>Bu da seçilir</p>
    </div>
</div>
Doğrudan Çocuk Seçici (>)
Sadece direkt alt öğeleri seçer. Derinlerdeki öğeleri seçmez.
Örnek:
.parent > p { color: red; }
Seçer: Sadece .parent'in direkt çocuğu olan p'ler
Bitişik Kardeş Seçici (+)
Öğenin hemen sonrasında gelen kardeş öğesini seçer.
Örnek:
h1 + p { font-weight: bold; margin-top: 10px; }
Seçer: h1'den hemen sonra gelen ilk p etiketi
Genel Kardeş Seçici (~)
Öğenin sonrasında gelen TÜM kardeş öğeleri seçer.
Örnek:
.active ~ .item { opacity: 0.5; }
Öznitelik Seçicileri ([attr])
HTML öznitelikleri (attributes) temelinde seçim yapar.
[attr]: Öznitelik var mı?
[disabled] { opacity: 0.5; }
[attr="value"]: Tam eşleşme
input[type="text"] { border: 1px solid #ccc; }
[attr^="value"]: Başlangıçta eşleşme
a[href^="https"] { color: green; }
[attr$="value"]: Sonunda eşleşme
img[src$=".svg"] { width: 24px; }
[attr*="value"]: Herhangi bir yerde eşleşme
a[href*="example"] { color: purple; }

🔄 Pseudo-class (:)

Öğenin durumuna göre stil uygulayan seçiciler

:hover
Fare öğenin üzerine geldiğinde uygulanır. Etkileşim geri bildirimi için idealdir.
Örnek:
button:hover { background-color: #059669; transform: scale(1.05); }
a:hover { color: #6366f1; text-decoration: underline; }
Canlı Demo:
:focus
Öğe odaklandığında (tıklandığında veya Tab ile seçildiğinde) uygulanır. Erişilebilirlik için önemlidir.
Örnek:
input:focus { outline: 2px solid #6366f1; border-color: #6366f1; }
button:focus { box-shadow: 0 0 0 3px rgba(99, 102, 241, 0.5); }
:active
Fare düğmesi basılı tutulduğu sırada uygulanır. Tıklama geri bildirimi için kullanılır.
Örnek:
button:active { transform: scale(0.98); background-color: #047857; }
a:active { color: #6366f1; }
:visited
Ziyaret edilmiş bağlantılar için stil. Başka öğelere uygulanamaz.
Örnek:
a:visited { color: #9333ea; }
📌 Güvenlik nedeniyle sınırlandırılmıştır
:first-child
Üst öğesinin ilk çocuğu ise seçer.
Örnek:
li:first-child { font-weight: bold; }
p:first-child { margin-top: 0; }
:last-child
Üst öğesinin son çocuğu ise seçer.
Örnek:
li:last-child { border-bottom: none; }
.card:last-child { margin-right: 0; }
:nth-child(n)
Belirli bir pozisyondaki çocukları seçer. Çok esnek ve güçlü bir seçicidir.
:nth-child(2)
Sadece 2. çocuk
:nth-child(odd)
Tek sayılı çocuklar
:nth-child(even)
Çift sayılı çocuklar
:nth-child(3n+1)
Her 3.'den 1 başlayarak
Örnek:
tr:nth-child(even) { background-color: #f3f4f6; }
.grid-item:nth-child(3n) { margin-right: 0; }
:not(selector)
Belirtilen seçici HARIÇ tüm öğeleri seçer. Olumsuz seçim için.
Örnek:
li:not(.active) { opacity: 0.5; }
input:not([type="submit"]) { padding: 10px; }
:is(selector)
Birden fazla seçiciden BİRİNİ eşleşen öğeleri seçer. Specificity'yi düşük tutar.
Örnek:
:is(h1, h2, h3) { color: #333; margin-bottom: 1rem; }
:is(.btn, button) { padding: 10px 20px; }
:where(selector)
:is() gibi ama 0 specificity ile çalışır. Kütüphane kodları için idealdir.
Örnek:
:where(.container) p { color: blue; }
:has(selector)
İçinde belirli öğeler barındıran üst öğeleri seçer. Ön-biçem için devrim.
Örnek:
.card:has(img) { padding: 10px; }
article:has(> h1) { border-top: 3px solid; }

✨ Pseudo-element (::)

Sanal öğeler oluşturarak dekoratif efektler ekleyin

::before
Öğenin BAŞINA sanal içerik ekler. content özelliği zorunludur.
Örnek:
.quote::before { content: '"'; color: #6366f1; font-size: 2em; }
Canlı Demo:
❝

Güzel bir alıntı

::after
Öğenin SONUNA sanal içerik ekler. Dekoratif çizgiler, ikonlar vb.
Örnek:
.required::after { content: " *"; color: red; }
::first-line
Metnin ilk satırını seçer. display:block olan öğelerde çalışır.
Örnek:
p::first-line { font-weight: bold; color: #6366f1; }
::first-letter
Metnin ilk harfini seçer. Drop cap (ilk harf büyültme) efekti için ideal.
Örnek:
p::first-letter { font-size: 2em; font-weight: bold; color: #6366f1; }
::placeholder
Input veya textarea placeholder metnini stiller.
Örnek:
input::placeholder { color: #999; font-style: italic; }
::selection
Fare ile seçilen metni stiller. color, background, text-shadow kullanabilirsiniz.
Örnek:
::selection { background-color: #6366f1; color: white; }
p::selection { background: #fbbf24; color: #111; }

📦 Box Model

Her HTML öğesi bir kutu (box) gibi davranır. Content, Padding, Border, Margin olmak üzere 4 katmandan oluşur.

width
Öğenin genişliğini belirler.
Değerler:
px (piksel) | em | rem | % | vw | auto | 100px
Örnek:
.box { width: 300px; }
.container { width: 100%; }
.sidebar { width: 25vw; }
height
Öğenin yüksekliğini belirler.
Örnek:
.hero { height: 500px; }
.card { height: auto; }
min-width, max-width, min-height, max-height
Öğenin minimum ve maksimum boyutlarını belirler. Responsive tasarım için kritiktir.
Örnek:
.container { min-width: 320px; max-width: 1200px; margin: 0 auto; }
.image { max-width: 100%; height: auto; }
margin
Öğenin DIŞ mesafesini (başka öğelerle arası) belirler. Kutunun etrafında boşluk yaratır.
Kısa yazım:
margin: 20px; (4 yön) margin: 20px 30px; (üst/alt, sağ/sol) margin: 10px 20px 30px 40px; (üst, sağ, alt, sol)
Uzun yazım:
margin-top, margin-right, margin-bottom, margin-left
Örnek:
.card { margin: 20px; }
h1 { margin-bottom: 1rem; }
.centered { margin: 0 auto; } /* Yatay ortalama */
padding
Öğenin İÇ mesafesini belirler. Content ile border arasında boşluk yaratır.
Yazım Margin ile aynıdır:
padding: 20px; padding: 10px 20px; padding-top, padding-right, padding-bottom, padding-left
Örnek:
.button { padding: 10px 20px; }
.container { padding: 40px; }
.card { padding: 20px 30px; }
border
Öğenin çevresine çerçeve (kenarlık) ekler.
border: [genişlik] [stil] [renk];
Stiller: solid | dashed | dotted | double | groove | ridge | inset | outset
Örnek:
.box { border: 2px solid #6366f1; }
.card { border: 1px dotted #ccc; }
.danger { border-bottom: 3px solid red; }
border-radius
Köşeleri yuvarlar. Modern tasarımın anahtar özelliği.
Değerler:
border-radius: 10px; (tüm köşeler) border-radius: 10px 20px; (değişken köşeler) border-radius: 50%; (tam daire)
Örnek:
.card { border-radius: 8px; }
.avatar { border-radius: 50%; }
.button { border-radius: 25px; }
Canlı Demo:
0px
8px
20px
50%
box-sizing
Genişlik ve yüksekliğin, padding ve border'ı içerip içermediğini belirler.
content-box (varsayılan):
width = sadece content
border-box (önerilir):
width = content + padding + border
Örnek (sıfırlama):
* { box-sizing: border-box; }
outline
Border'ın dışında bir çizgi ekler. Erişilebilirlik için önemlidir (focus durumunda).
Örnek:
button:focus { outline: 2px solid #6366f1; outline-offset: 2px; }

✍️ Typography

Yazı (metin) özelliklerini kontrol edin. İyi tipografi, tasarımın %90'ıdır!

font-family
Metnin yazı tipini (font) belirler. Yedek yazı tipleri için virgülle ayırılır.
Sistem fontları:
Arial, Helvetica, Georgia, Times New Roman, Courier New
Web fontları:
'Inter', 'Poppins', 'Roboto', 'Playfair Display'
Örnek:
body { font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif; }
.serif { font-family: Georgia, 'Times New Roman', serif; }
font-size
Metnin büyüklüğünü belirler. px, em, rem, % kullanılabilir.
Birimler:
px (piksel - mutlak) | em (ebeveynin boyutu) | rem (kök boyutu) | % (yüzde)
Örnek:
body { font-size: 16px; }
h1 { font-size: 2rem; }
.small { font-size: 0.875em; }
font-weight
Metnin kalınlığını (inceliğini) belirler. Yazı tipinin desteklediği ağırlıklar vardır.
Değerler:
normal (400) | bold (700) | lighter | bolder
Sayı değerleri:
100 (Thin) | 300 (Light) | 400 (Normal) | 500 (Medium) | 700 (Bold) | 900 (Black)
Örnek:
h1 { font-weight: 700; }
.light-text { font-weight: 300; }
strong { font-weight: bold; }
font-style
Metni italik veya normal yapar.
Değerler:
normal | italic | oblique
Örnek:
em { font-style: italic; }
.quote { font-style: italic; color: #666; }
line-height
Satırlar arasındaki mesafe. Okunabilirlik için çok önemlidir.
Değerler:
normal (1.2) | sayı (1.5) | px (20px) | % (120%)
İdeal değer:
1.5 - 1.8 arasında (body metni için)
Örnek:
body { line-height: 1.6; }
h1 { line-height: 1.2; }
.article p { line-height: 1.8; }
letter-spacing
Karakterler arasındaki mesafe. Başlıklar için stil için kullanılır.
Örnek:
.title { letter-spacing: 2px; }
.logo { letter-spacing: 3px; text-transform: uppercase; }
word-spacing
Sözcükler arasındaki mesafe.
Örnek:
.spaced-text { word-spacing: 5px; }
text-align
Metni hizalar.
Değerler:
left | center | right | justify
Örnek:
h1 { text-align: center; }
p { text-align: justify; }
text-decoration
Metne altçizgi, üstçizgi vb. ekler.
Değerler:
none | underline | overline | line-through
Örnek:
a { text-decoration: none; }
del { text-decoration: line-through; }
ins { text-decoration: underline; }
text-transform
Metnin büyüklüğünü değiştirir (HTML'i değiştirmeden).
Değerler:
none | uppercase | lowercase | capitalize
Örnek:
.uppercase { text-transform: uppercase; }
.title { text-transform: capitalize; }
text-shadow
Metne gölge efekti ekler.
Yazım:
text-shadow: [x] [y] [blur] [renk];
Örnek:
.title { text-shadow: 2px 2px 4px rgba(0,0,0,0.3); }
.neon { text-shadow: 0 0 10px #6366f1, 0 0 20px #6366f1; }
Google Fonts Kullanımı
İnternet üzerinden ücretsiz yazı tipleri yükleyin. fonts.google.com'dan seçin.
HTML <head> içinde:
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">
CSS'de:
body { font-family: 'Inter', sans-serif; }
@font-face
Kendi yazı dosyalarınızı yükleyin. Özel (custom) fontlar için kullanılır.
Örnek:
@font-face {
    font-family: 'CustomFont';
    src: url('fonts/custom.woff2') format('woff2');
    font-weight: normal;
    font-style: normal;
}

🎨 Renkler & Arka Plan

Renk ve arka plan özelliklerini kontrol edin. Tasarımın ruh ve hissi renkte yatar.

color
Metin rengini belirler.
Renk Formatları:
Hex: #FF0000 | RGB: rgb(255, 0, 0) | HSL: hsl(0, 100%, 50%) | Renk İsmi: red, blue
Örnek:
p { color: #333333; }
.warning { color: rgb(255, 152, 0); }
a { color: hsl(240, 100%, 50%); }
background-color
Öğenin arka plan rengini belirler.
Örnek:
.card { background-color: #ffffff; }
.highlight { background-color: rgba(255, 193, 7, 0.5); }
background-image
Arka plan olarak resim veya gradient ekler.
Örnek:
.hero { background-image: url('bg.jpg'); }
.gradient { background-image: linear-gradient(135deg, #667eea 0%, #764ba2 100%); }
background-size
Arka plan resminin boyutunu belirler.
Değerler:
cover (kopyala) | contain (sığdır) | 100px | 50% | auto
Örnek:
.hero { background-size: cover; background-position: center; }
.pattern { background-size: 50px 50px; }
background-position
Arka plan resminin konumunu belirler.
Örnek:
.hero { background-position: center; }
.banner { background-position: center top; }
background-repeat
Arka plan resminin tekrar edip etmediğini belirler.
Değerler:
repeat | no-repeat | repeat-x | repeat-y
Örnek:
.pattern { background-repeat: repeat; }
.hero { background-repeat: no-repeat; }
linear-gradient()
İki veya daha fazla renk arasında doğrusal geçiş yapar.
Yazım:
linear-gradient([yön], renk1, renk2, ...)
Yönler:
to right, to bottom, 135deg (açı), vb.
Örnek:
.header { background: linear-gradient(to right, #6366f1, #10b981); }
.diagonal { background: linear-gradient(135deg, red, yellow); }
Canlı Demo:
radial-gradient()
Merkezi bir noktadan dışa doğru renk geçişi yapar.
Örnek:
.spotlight { background: radial-gradient(circle, yellow, red); }
.fade { background: radial-gradient(ellipse at center, #fff, #000); }
conic-gradient()
Dönel (huni şeklinde) renk geçişi. Pasta grafikleri için ideal.
Örnek:
.pie { background: conic-gradient(red 25%, yellow 25% 50%, green 50%); }
opacity
Öğenin saydamlığını (şeffaflığını) belirler. 0 (tamamen saydamdır) ile 1 (opak) arasında.
Örnek:
.ghost { opacity: 0.5; }
.watermark { opacity: 0.2; }
img:hover { opacity: 1; }
Renk Formatları Karşılaştırması
Farklı renk tanımlama yöntemlerini karşılaştırın.
Örnek Renk (Kırmızı):
Hex: #FF0000
RGB: rgb(255, 0, 0)
HSL: hsl(0, 100%, 50%)
İsim: red

👁️ Display & Görünürlük

Öğelerin sayfada nasıl gösterildiğini kontrol edin.

display: block
Öğe yeni satırdan başlar ve tam genişliği kaplar. p, div, h1-h6 varsayılan olarak block'tur.
Örnek:
div { display: block; }
.full-width { display: block; width: 100%; }
display: inline
Öğe metinle birlikte aynı satırda yer alır. width ve height çalışmaz. span, a varsayılan olarak inline'dır.
Örnek:
strong { display: inline; }
.tag { display: inline; margin-right: 5px; }
display: inline-block
Block ve inline'ın karışımı. Aynı satırda yer alır ama width/height ve padding/margin çalışır.
Örnek:
.button { display: inline-block; padding: 10px 20px; margin-right: 10px; }
display: none
Öğe seçini tamamen kaldırır. Sayfada görünmez ve yer kaplamaz.
Örnek:
.mobile-only { display: none; }
@media (max-width: 768px) { .mobile-only { display: block; } }
display: flex
Flexbox layout'u etkinleştirir. Hizalamayı çok kolay hale getirir.
Örnek:
.container { display: flex; gap: 20px; }
.navbar { display: flex; justify-content: space-between; align-items: center; }
display: grid
CSS Grid layout'u etkinleştirir. 2D düzenleme için müthiş bir araçtır.
Örnek:
.grid { display: grid; grid-template-columns: 1fr 1fr 1fr; gap: 20px; }
visibility
Öğeyi gösterin/gizleyin. display:none'dan farklı olarak yer yine kaplar.
Değerler:
visible (göster) | hidden (gizle) | collapse
Örnek:
.hidden { visibility: hidden; } /* Yer hala kaplar */
.visible { visibility: visible; }
overflow
İçerik kutunun sınırlarından taşarsa ne yapılacağını belirler.
Değerler:
visible (göster) | hidden (kırp) | scroll (kaydırma çubuğu ekle) | auto (gerekirse kaydırma)
Ayrı ayrı:
overflow-x | overflow-y
Örnek:
.box { width: 300px; height: 200px; overflow: auto; }
.container { overflow-x: hidden; overflow-y: scroll; }
clip-path
Öğeyi belirli bir şekille kırpar. Yaratıcı şekiller oluşturur.
Örnek:
.circle { clip-path: circle(50%); }
.triangle { clip-path: polygon(0 0, 100% 0, 50% 100%); }
.image { clip-path: inset(10% 20% 30% 40%); }

💪 Flexbox

1D (tek boyutlu) düzenleme için güçlü ve esnek bir araçtır. Navigasyon, kartlar, form elemanları için idealdir.

display: flex
Öğeyi flex container'a dönüştürür. Çocuklarına güçlü hizalama seçenekleri verir.
Örnek:
.container { display: flex; }
flex-direction
Çocuk öğelerin yönünü belirler.
Değerler:
row (sol-sağ) | column (üst-alt) | row-reverse | column-reverse
Örnek:
.navbar { display: flex; flex-direction: row; }
.sidebar { display: flex; flex-direction: column; }
justify-content
Ana eksen (row ise yatay, column ise dikey) üzerinde öğeleri hizalar.
Değerler:
flex-start (başlangıç) | flex-end (bitiş) | center (orta) | space-between (aralar açık) | space-around (etraf açık) | space-evenly (eşit)
Örnek:
.header { display: flex; justify-content: space-between; }
.centered { display: flex; justify-content: center; }
align-items
Çapraz eksen (row ise dikey, column ise yatay) üzerinde öğeleri hizalar.
Değerler:
flex-start | flex-end | center | stretch | baseline
Örnek:
.card { display: flex; align-items: center; }
.banner { display: flex; align-items: flex-start; }
gap
Çocuk öğeler arasındaki boşluğu belirler. Margin'den daha esnek.
Örnek:
.grid { display: flex; gap: 20px; }
.flex { display: flex; gap: 10px 20px; } /* row gap, column gap */
flex-wrap
Çocuklar yer yetersiz olduğunda sıraya başlangıç mi, yoksa taşması mı gerekir?
Değerler:
nowrap (sarmama) | wrap (sarmama) | wrap-reverse
Örnek:
.flex { display: flex; flex-wrap: wrap; gap: 10px; }
flex-grow, flex-shrink, flex-basis
Çocuk öğelerin boyut davranışını kontrol eder.
flex-grow:
Boş alan paylaşı (0-1 arası)
flex-shrink:
Daralma oranı
flex-basis:
Başlangıç boyutu
Kısa:
flex: grow shrink basis
Örnek:
.item { flex: 1; } /* Eşit büyülüş */
.sidebar { flex: 0 0 300px; } /* Sabit genişlik */
.main { flex: 1 1 0; } /* Esnek */

🎯 CSS Grid

2D (iki boyutlu) düzenleme için en güçlü CSS özelliği. Web sayfalarını tam olarak kontrol edin.

display: grid
Öğeyi grid container'a dönüştürür.
Örnek:
.container { display: grid; }
grid-template-columns
Sütun sayısını ve genişliklerini belirler.
Örnek:
.grid { grid-template-columns: 1fr 1fr 1fr; } /* 3 eşit sütun */
.layout { grid-template-columns: 300px 1fr 200px; } /* Sabit, esnek, sabit */
.responsive { grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); }
grid-template-rows
Satır sayısını ve yüksekliklerini belirler.
Örnek:
.page { grid-template-rows: auto 1fr auto; } /* Header, Main, Footer */
gap / grid-gap
Satır ve sütun arasındaki boşluk.
Örnek:
.grid { gap: 20px; } /* Tüm taraflar */
.grid { gap: 20px 30px; } /* Satır 20px, Sütun 30px */
grid-column, grid-row
Çocuk öğelerin kaç sütun/satır kaplayacağını belirler.
Örnek:
.hero { grid-column: 1 / -1; } /* Tüm genişlik */
.sidebar { grid-column: 1 / 2; grid-row: 1 / 3; }
fr (Fraction) Birimi
Kalan alanı orantılı olarak böler. Grid'in kalbidir.
Örnek:
.grid { grid-template-columns: 1fr 1fr 1fr; } /* Eşit */
.layout { grid-template-columns: 1fr 2fr 1fr; } /* 1:2:1 oranı */
repeat() Fonksiyonu
Tekrar eden grid tanımlarını kısaltır.
Örnek:
.grid { grid-template-columns: repeat(3, 1fr); }
.flex { grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); }

📍 Pozisyonlama

Öğeleri sayfada tam istediğiniz yere yerleştirin.

position: static
Varsayılan konum. Belge akışı içinde yer alır. top, right vb. çalışmaz.
Örnek:
div { position: static; } /* Varsayılan */
position: relative
Normal konumundan göre yer değiştirir. Yer yine kaplar.
Örnek:
.box { position: relative; top: 10px; left: 20px; }
position: absolute
Belge akışından çıkar. En yakın positioned (position:relative vb.) üst öğesine göre konumlanır.
Örnek:
.container { position: relative; }
.badge { position: absolute; top: 0; right: 0; }
position: fixed
Ekrana göre sabittir. Sayfanın diğer kısımları kaydırıldığında hareketsiz kalır.
Örnek:
.navbar { position: fixed; top: 0; left: 0; right: 0; }
.chat-btn { position: fixed; bottom: 20px; right: 20px; }
position: sticky
Kayan (floating) öğe. Normal aktış izler, ardından sabitlenir.
Örnek:
.header { position: sticky; top: 0; }
.sidebar { position: sticky; top: 20px; height: calc(100vh - 20px); }
z-index
Öğelerin üst üste sırasını belirler. Yüksek değer öne gelir. Position relative/absolute/fixed gereklidir.
Örnek:
.modal { position: fixed; z-index: 1000; }
.backdrop { position: fixed; z-index: 999; }
.tooltip { position: absolute; z-index: 10; }
top, right, bottom, left
Konumlandırılmış öğelerin kenar mesafelerini belirler.
Örnek:
.modal { position: fixed; top: 50%; left: 50%; transform: translate(-50%, -50%); }
.corner { position: absolute; top: 0; right: 0; }
float
Öğeyi sola/sağa hizalar. Metni otomat olarak etrafında akar. (Eski yöntem, Flexbox tercihlidir)
Değerler:
none | left | right
Örnek (eski):
.image { float: left; margin-right: 20px; }
clear
Float öğeleri temizler. (Eski yöntem)
Örnek:
.footer { clear: both; }

✨ Geçişler & Animasyonlar

Harekete ve heyecana dönüştürün! Tasarımınızı canlı hale getirin.

transition
Değişikleri düzgün bir şekilde geçiş yapar. Basit animasyonlar için idealdir.
Yazım:
transition: [property] [duration] [timing-function] [delay];
Örnek:
.button { background: blue; transition: background 0.3s ease; }
.button:hover { background: darkblue; }
transition-property, transition-duration, transition-timing-function, transition-delay
Geçişi kontrol etmek için ayrı ayrı özellikler.
Örnek:
.card {
    transition-property: transform, box-shadow;
    transition-duration: 0.3s;
    transition-timing-function: cubic-bezier(0.4, 0, 0.2, 1);
    transition-delay: 0.1s;
}
@keyframes
Özel animasyonlar tanımlar. Başlangıç (0%) ve bitiş (100%) arasında adımlar belirtirsiniz.
Örnek:
@keyframes fadeIn {
    0% { opacity: 0; }
    100% { opacity: 1; }
}
@keyframes slide {
    0% { transform: translateX(-100px); }
    100% { transform: translateX(0); }
}
animation
@keyframes animasyonunu uygulatır.
Yazım:
animation: [name] [duration] [timing] [delay] [iteration] [direction];
Örnek:
.box { animation: fadeIn 1s ease-in-out; }
.bounce { animation: jump 0.5s ease-out infinite alternate; }
transform
Öğeyi dönüş, ölçek, kaydırma ile değiştir. GPU hızlandırmalı, verimli.
Değerler:
translate() | rotate() | scale() | skew() | matrix()
Örnek:
.button:hover { transform: scale(1.05); }
.spin { animation: rotate 2s linear infinite; }
@keyframes rotate { to { transform: rotate(360deg); } }
Timing Functions
Animasyonun hızını kontrol eden eğriler.
Değerler:
linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier()

📱 Responsive Design

Her ekran boyutunda güzel görünen tasarımlar oluşturun.

@media Queries
Ekran boyutuna göre farklı CSS kuralları uygular. Responsive tasarımın temelini oluşturur.
Örnek:
/* Mobil (varsayılan) */
.container { width: 100%; padding: 20px; }

/* Tablet */
@media (min-width: 768px) {
    .container { width: 750px; margin: 0 auto; }
}

/* Masaüstü */
@media (min-width: 1024px) {
    .container { width: 960px; }
}
Viewport Meta Etiketi
Mobil cihazlar için ölçeği ayarlar. MUTLAKA header'a ekleyin!
HTML <head> içinde:
<meta name="viewport" content="width=device-width, initial-scale=1.0">
min-width vs max-width
Responsive davranışı kontrol eder.
min-width:
"Bu boyuttan BÜYÜKSE uygula"
max-width:
"Bu boyuttan KÜÇÜKSEThen uygula"
Örnek:
/* Mobil cihazlar için */
@media (max-width: 767px) { ... }

/* Tablet ve üzeri */
@media (min-width: 768px) { ... }

/* Büyük ekranlar */
@media (min-width: 1200px) { ... }
Viewport Birimleri (vw, vh, vmin, vmax)
Ekran boyutuna göre dinamik ölçüler.
vw (Viewport Width):
Ekran genişliğinin % oranı
vh (Viewport Height):
Ekran yüksekliğinin % oranı
vmin:
vw ve vh'den küçük olanı
vmax:
vw ve vh'den büyük olanı
Örnek:
.hero { height: 100vh; } /* Full ekran */
.title { font-size: 5vw; } /* Ekran genişliğine göre */
.box { width: 50vmin; height: 50vmin; } /* Kare, ekrana göre
em ve rem Birimleri
Göreceli ölçüler. Responsive ve esnek tasarım için idealdir.
em:
Ebeveyn öğenin font-size'ına göreceli
rem:
Kök (<html>) font-size'ına göreceli (önerilir)
Örnek:
html { font-size: 16px; }
body { font-size: 1rem; } /* 16px */
h1 { font-size: 2rem; } /* 32px */
.small { font-size: 0.875rem; } /* 14px */
clamp() Fonksiyonu
Minimum ve maksimum arasında responsif değer. Fluid typography için muhteşem.
Yazım:
clamp(MIN, PREF, MAX)
Örnek:
h1 { font-size: clamp(1.5rem, 5vw, 3rem); }
.container { width: clamp(300px, 90%, 1200px); }
min() ve max() Fonksiyonları
En küçük/büyük değeri seçer. Responsive bir çok durumda kullanılır.
Örnek:
.box { width: min(100%, 500px); } /* Maksimum 500px, ama daha az olabilir */
.padding { padding: max(1rem, 5vw); } /* Minimum 1rem */

🚀 Modern CSS

CSS'in en yeni ve en güçlü özellikleri. Geleceği şimdi inşa edin!

CSS Değişkenleri (Custom Properties)
Tekrar kullabilir değerler tanımlayın. Tema değişmesi çok kolay hale gelir.
Örnek:
:root {
    --primary: #6366f1;
    --secondary: #10b981;
    --spacing: 1rem;
    --radius: 8px;
}

.button {
    background: var(--primary);
    padding: var(--spacing);
    border-radius: var(--radius);
}
calc() Fonksiyonu
Matematiksel işlemler yapar. Responsive tasarımda vazgeçilmez.
Örnek:
.container { width: calc(100% - 40px); }
.gap { margin: calc(1rem + 2vw); }
.aspect { height: calc(100vw * 9 / 16); } /* 16:9 aspect ratio */
filter
Görsel efektler ekler: bulanıklık, parlaklık, kontrast vb.
Değerler:
brightness() | contrast() | blur() | grayscale() | sepia() | hue-rotate() | invert() | saturate() | drop-shadow()
Örnek:
.image:hover { filter: brightness(1.2) contrast(1.1); }
.grayscale { filter: grayscale(100%); }
.shadow { filter: drop-shadow(0 4px 8px rgba(0,0,0,0.2)); }
backdrop-filter
Arkaplanda filter uygular. Cam blur (frosted glass) efekti için ideal.
Örnek:
.modal { backdrop-filter: blur(5px); }
.navbar { backdrop-filter: blur(10px); background: rgba(255,255,255,0.8); }
mix-blend-mode
Öğeyi altındaki öğeyle nasıl karıştıracağını belirler. Tasarımda yaratıcı efektler için.
Değerler:
multiply | screen | overlay | darken | lighten | color-dodge | vb.
Örnek:
.image { mix-blend-mode: multiply; }
.overlay { mix-blend-mode: screen; }
aspect-ratio
Sabit en-boy oranı. Video ve görsel konteynerleri için mükemmel.
Örnek:
.video { aspect-ratio: 16 / 9; width: 100%; height: auto; }
.square { aspect-ratio: 1; width: 200px; }
Container Queries
Viewport yerine konteyner boyutuna göre stil uygula. Bileşen tabanlı tasarımda devrim!
Örnek:
@container (min-width: 400px) {
    .card { display: grid; grid-template-columns: 1fr 1fr; }
}
⚠️ Chrome 105+, Safari 16+
:has() Selector (Parent Selector)
İçinde belirli öğeler olan üst öğeleri seçer. CSS'de ana öğeyi seçme artık mümkün!
Örnek:
.card:has(img) { padding: 10px; }
.item:has(input:checked) { background: #f0f0f0; }
CSS Nesting
Saçsan CSS kurallarını iç içe yazma. Sass/LESS gibi bileşik stil yazma!
Örnek:
.card {
    padding: 20px;

    &:hover {
        box-shadow: 0 10px 20px rgba(0,0,0,0.1);
    }

    p {
        margin: 0 0 10px 0;
    }
}
⚠️ Tarayıcı desteği artıyor (2024+)

⚖️ Specificity & Cascade

CSS kurallarının hangisinin uygulanacağını belirleyen kurallar. "Açık stilin sırrı"!

Specificity (Belirlilik)
Hangi CSS kuralının uygulanacağını belirler. Daha spesifik olan kazanır.
Sıra (düşükten yükseğe):
1. Eleman seçici (1 puan) 2. Sınıf seçici (10 puan) 3. ID seçici (100 puan) 4. İnline stil (1000 puan) 5. !important (10000 puan - son çare!)
Örnek:
p { color: blue; } /* 1 puan */
.highlight { color: red; } /* 10 puan (kazanır) */
#special { color: green; } /* 100 puan (kazanır) */
<p style="color: yellow;"> /* 1000 puan (kazanır) */
Cascade (Basamak)
Aynı specificity'de, SON kural kazanır. Dosyanın sırası önemlidir.
Örnek:
.box { color: blue; }
.box { color: red; } /* ← Bu uygulanır */
Inheritance (Kalıtım)
Bazı CSS özellikleri ebeveynden çocuğa aktarılır.
Kalıtım YAPAN özellikler:
color, font-family, font-size, line-height, text-align
Kalıtım YAPMAYAN özellikler:
width, height, margin, padding, border, background
Örnek:
.container { color: blue; } /* Tüm çocuklar blue metne sahip */
.container { padding: 20px; } /* Çocuklar bu padding'i almaz */
!important Kuralı
Her şeyi geçersiz kılar. SADECE SON ÇARE olarak kullanın!
Örnek:
.primary { background: blue !important; }
⚠️ Kötü bir alışkanlıktır! Arkaplan kod bakım yapmayı zorlaştırır
