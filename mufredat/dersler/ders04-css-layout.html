Flexbox (Esnek Kutu) 📦

Flexbox, bir boyutlu (satır veya sütun) düzenler için mükemmeldir. Elemanlar arası boşluğu ve hizalamayı kolayca kontrol edebilirsiniz.

Flexbox Neden Önemli?

  • Responsive tasarım için ideal
  • Merkezi hizalama çok kolay
  • Dinamik boyutlandırma
  • Orijinal sıra değiştirilebilir

Ana Flexbox Özellikleri

Özellik Açıklama Değerler
display: flex Flexbox aktivesi flex, inline-flex
flex-direction Yön (satır/sütun) row, column, row-reverse
justify-content Ana eksen hizalama flex-start, center, space-between
align-items Çapraz eksen hizalama flex-start, center, stretch
flex-wrap Sarma davranışı nowrap, wrap, wrap-reverse
gap Elemanlar arası boşluk 10px, 1em, vb.
flex (item) Eleman büyüme/küçülme 1, 0 1 auto, vb.

Flexbox Örneği

Merkezi Hizalanmış Flexbox

.container { display: flex; justify-content: center; /* Yatay merkez */ align-items: center; /* Dikey merkez */ height: 200px; gap: 20px; } .item { width: 100px; padding: 20px; background: #6366f1; color: white; border-radius: 8px; }

CSS Grid 🎴

Grid, iki boyutlu (satır ve sütun) düzenler için yapılmıştır. Kompleks sayfaları çok kolay bir şekilde organize edebilirsiniz.

Grid vs Flexbox

Özellik Flexbox Grid
Boyut 1 boyutlu 2 boyutlu
Kullanım Basit düzenler Kompleks düzenler
Hizalama Ana eksen Her iki eksen

Grid Temel Özellikleri

Özellik Açıklama
display: grid Grid aktivesi
grid-template-columns Sütun tanımlama
grid-template-rows Satır tanımlama
gap Elemanlar arası boşluk
grid-column Sütun yayılması
grid-row Satır yayılması

Grid Örneği

3 Sütunlu Grid

.container { display: grid; grid-template-columns: repeat(3, 1fr); gap: 20px; } .item { padding: 30px; background: #10b981; color: white; border-radius: 8px; text-align: center; font-weight: 600; }

Pozisyonlama 📍

position özelliği, bir elemanın sayfada nereye konumlandırılacağını belirler.

Değer Açıklama
static Normal akış (varsayılan)
relative Normal konumdan göreceli olarak kaydırma
absolute En yakın konumlandırılmış ebeveynine göre
fixed Ekran sabitlenmiş (kaydırıldığında sabit kalır)
sticky Kaydırmaya kadar relative, sonra fixed

Position Türleri Örneği

Relative

Top: 20px, Left: 30px

Absolute

ABSOLUTE
Sağ üst köşede konumlandırılmış

Position Kod Örneği

Position Uygulaması

/* Ebeveyn relative olmalı */ .container { position: relative; } /* Mutlak konumlandırma */ .absolute-item { position: absolute; top: 10px; right: 10px; width: 50px; height: 50px; background: #ef4444; } /* Ekran sabitlenmiş */ .floating-button { position: fixed; bottom: 20px; right: 20px; z-index: 1000; }

Position örnekleri yukarıda gösterilmiştir

Z-Index (Katmanlaşma)

z-index, elemanların örtüşmesi durumunda hangisinin üstü geleceklerini belirler. Daha yüksek değer, daha üstte.

Z-Index Örneği

.overlay { position: fixed; z-index: 999; } .modal { position: fixed; z-index: 1000; /* Overlay'ın üstünde */ } .tooltip { position: absolute; z-index: 10; }

Daha yüksek z-index = Daha üst katman

Alıştırmalar 💪

🎯 Alıştırma 1: Flexbox menüsü oluşturun:
  • Yatay menü oluşturun (flex-direction: row)
  • Logoyu sola, menüyü sağa yerleştirin
  • space-between kullanarak boşluk oluşturun
  • Hover efektleri ekleyin
🎯 Alıştırma 2: Grid galerisi oluşturun:
  • 3 sütunlu responsive galeri yapın
  • auto-fit veya auto-fill kullanın
  • Resimlere aspect-ratio uygulayın
  • Hover'da zoom efekti ekleyin
🎯 Alıştırma 3: Sayfayı yerleşik yapın:
  • Sticky header oluşturun
  • Fixed floating action button ekleyin
  • Position özelliklerini test edin
🎯 Alıştırma 4: Kompleks sayfa tasarımı:
  • Grid ile sayfanın genel düzenini yapın
  • Header, Sidebar, Main, Footer oluşturun
  • Main alanında kart düzeni için Flexbox/Grid kullanın
  • Tamamen responsive hale getirin
