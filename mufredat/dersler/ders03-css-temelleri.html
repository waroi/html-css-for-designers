CSS Nedir? 🎨

CSS (Cascading Style Sheets), HTML sayfalarını stilize etmek ve tasarlamak için kullanılan dildir. HTML yapıyı sağlar, CSS ise görünümü sağlar.

CSS'in "Cascading" kelimesi, stillerin aşağıdan yukarıya uygulanması anlamına gelir. Daha sonra yazılan kurallar, daha önce yazılan kuralları geçersiz kılabilir.

💡 Designer İpucu: HTML içerik, CSS ise sunum! Bunları birbirinden ayrı tutmak çok önemlidir.

CSS Neden Önemli?

  • Tasarım: Sayfanın görünümünü kontrol eder
  • Tutarlılık: Tüm sayfalara aynı stil uygular
  • Yönetim: Tek dosyada tüm stilleri toplar
  • Performans: Sayfaları hızlandırır
  • Responsive: Farklı cihazlara uyum sağlar

CSS Bağlama Yöntemleri 🔗

HTML'e CSS eklemek için 3 farklı yöntem vardır:

1. Dış CSS Dosyası (Önerilen)

Dış CSS Bağlama

<!-- HTML Dosyasında --> <head> <link rel="stylesheet" href="stil.css"> </head> /* stil.css dosyasında */ body { background-color: #f0f0f0; color: #333; } h1 { color: #6366f1; }

✓ En İyi Pratik

Ayrı dosya, kolay yönetim

2. İç CSS (Style Etiketi)

Style Etiketi Örneği

<head> <style> body { background-color: #f0f0f0; } h1 { color: #6366f1; } </style> </head>

⚠️ Orta Uyumluluk

HTML dosyasında tanımlanır

3. Satır İçi CSS (Inline)

Inline Stil Örneği

<h1 style="color: #6366f1; font-size: 32px;"> Merhaba Dünya </h1> <p style="background-color: yellow;"> Sarı arka plan </p>

✗ Kaçınılmalı

Zorlayıcı, yönetimi zor

⚠️ Dikkat: Dış CSS dosyası kullanmak en iyi pratiktir! Satır içi CSS'i mümkün olduğunca kaçının.

CSS Seçicileri 🎯

Seçiciler, stilini uygulamak istediğiniz HTML elemanlarını belirtir.

Temel Seçiciler

Seçici Açıklama Örnek
* Tüm elemanlar * { margin: 0; }
etiket Etiket seçici p { color: blue; }
.class Class seçici .baslik { font-size: 20px; }
#id ID seçici #logo { width: 100px; }
etiket.class Etiket + Class p.kirmizi { color: red; }
etiket > alt Direkt alt eleman ul > li { margin: 5px; }
etiket alt Tüm alt elemanlar div p { color: gray; }
:hover Fare üzerinde a:hover { color: red; }
:active Tıklanırken button:active { scale: 0.95; }

Seçici Özgüllüğü (Specificity)

Birden fazla stil aynı elemana uygulanırsa, en özgül olan kazanır.

Seçici Tipi Özgüllük
Inline Style 1000
ID (#) 100
Class (.) ve Pseudo 10
Etiket 1

Seçici Örnekleri

/* Tüm p etiketleri */ p { color: blue; } /* .baslik class'ına sahip tüm elemanlar */ .baslik { font-size: 24px; font-weight: bold; } /* id="logo" olan eleman */ #logo { width: 200px; } /* Hover efekti */ a:hover { text-decoration: underline; color: red; }

Bu p etiketi mavi

Bu .baslik class'ı
Logo (ID)

Renk Sistemleri 🌈

CSS'de renkler üç farklı şekilde tanımlanabilir: HEX, RGB, HSL

HEX (Heksadesimal)

En sık kullanılan yöntem. # işareti ile başlar.

#FF0000 (Kırmızı)
#00FF00 (Yeşil)
#0000FF (Mavi)

RGB (Red, Green, Blue)

0-255 arasında değer alır. Saydam renk için RGBA.

RGB Renk Örneği

/* RGB */ color: rgb(255, 0, 0); /* Kırmızı */ background: rgb(0, 255, 0); /* Yeşil */ /* RGBA (Saydam) */ background: rgba(100, 200, 255, 0.5); /* Son sayı: 0 = tamamen saydam, 1 = tamamen opak */
RGB(255,0,0)
RGBA(100,200,255,0.5)

HSL (Hue, Saturation, Lightness)

İnsan dostu renk sistemi. Ton, doygunluk, parlaklık.

HSL Renk Örneği

/* HSL */ color: hsl(0, 100%, 50%); /* Kırmızı */ background: hsl(120, 100%, 50%); /* Yeşil */ border-color: hsl(240, 100%, 50%); /* Mavi */ /* HSLA (Saydam) */ background: hsla(200, 80%, 50%, 0.7);
Kırmızı
Yeşil
Mavi

Tipografi 🔤

Metni stilize etmek, web sayfasının görünümü açısından çok önemlidir.

Font Özellikleri

Özellik Açıklama Örnek
font-family Yazı tipi Arial, Helvetica, sans-serif
font-size Yazı büyüklüğü 16px, 1.5em, 2rem
font-weight Yazı kalınlığı normal, bold, 700
font-style Eğik yazı normal, italic
line-height Satır yüksekliği 1.5, 1.8, 24px
letter-spacing Harf aralığı 2px, 0.05em
text-align Metin hizalama left, center, right, justify
text-decoration Metin dekoru none, underline, overline, line-through
color Metin rengi #333, rgb(50,50,50)

Tipografi Örneği

Stilize Metin

h1 { font-family: Georgia, serif; font-size: 36px; font-weight: bold; color: #333; line-height: 1.4; } p { font-family: 'Segoe UI', sans-serif; font-size: 16px; line-height: 1.6; color: #666; } a { color: #0066cc; text-decoration: none; } a:hover { text-decoration: underline; }

Stilize Başlık

Bu metin stilize edilmiş bir paragraf örneğidir. Font tipi, boyutu ve satır yüksekliği dikkatli seçilmiştir.

Bağlantı Örneği

Box Model 📦

CSS'deki tüm elemanlar birer kutu (box) olarak görülür. Her kutu içeriği, padding, border ve margin'den oluşur.

Box Model Parçaları

Margin (Dış Boşluk)
Border (Çerceve)
Padding (İç Boşluk)
Content (İçerik)

Box Model Örneği

Kutu Örneği

div { /* İçerik */ width: 200px; height: 100px; /* İç boşluk */ padding: 20px; /* Çerceve */ border: 2px solid #333; /* Dış boşluk */ margin: 15px; /* Arka plan */ background-color: #f0f0f0; }
Kutu İçeriği

Box Sizing

Box Sizing Türleri

/* content-box (Varsayılan) */ /* width = sadece içerik */ box-sizing: content-box; /* border-box (Önerilen) */ /* width = içerik + padding + border */ box-sizing: border-box; /* Tüm elemanlar için */ * { box-sizing: border-box; }

✓ border-box Kullanın!

Genişlik hesaplaması daha kolay

Display Özellikleri 🎭

display özelliği, bir elemanın sayfada nasıl davranacağını belirler.

Değer Açıklama Kullanım
block Tam satır kaplar div, p, h1-h6
inline Gerekli kadar yer kaplar span, a, strong
inline-block İkisinin kombinasyonu img, button
flex Flexbox layout Kompleks düzenler
grid Grid layout İki boyutlu düzenler
none Gizler (DOM'dan çıkar) Gizlenecek elemanlar

Display Örnekleri

Display Değerleri

/* Block */ div { display: block; width: 100%; } /* Inline */ span { display: inline; } /* Inline-Block */ img { display: inline-block; } /* Gizle */ .gizli { display: none; }
Block (Tam satır)
Inline (Gerekli kadar)
💡 Designer İpucu: Flexbox ve Grid, modern CSS'de en sık kullanılan display değerleridir. Bunları sonraki derslerde detaylı öğreneceğiz!

Alıştırmalar 💪

🎯 Alıştırma 1: Dış CSS dosyası ile bir sayfa tasarlayın:
  • Farklı başlık stilleri ekleyin
  • Paragraf stillerini ayarlayın
  • En az 5 farklı renkli eleman oluşturun
🎯 Alıştırma 2: Seçici ve özgüllük pratikleri:
  • Etiket, class ve ID seçicileri kullanın
  • Hover efektleri ekleyin
  • Çelişen stilleri çözmek için özgüllüğü kullanın
🎯 Alıştırma 3: Renkli bir palet oluşturun:
  • HEX, RGB, HSL'de renkler tanımlayın
  • Saydam renkler (RGBA) kullanın
  • Gradyan arka plan ekleyin (bonus)
🎯 Alıştırma 4: Box Model pratikleri:
  • Çeşitli padding ve margin değerleri deneyin
  • Border stillerini özelleştirin
  • Box-sizing özelliğini test edin
