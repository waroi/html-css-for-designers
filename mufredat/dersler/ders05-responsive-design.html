Responsive Design Nedir? 📱

Responsive Design, web sayfalarının tüm cihaz boyutlarına uyum sağlamasıdır. Telefondan tablete, masaüstüne kadar her ekranda mükemmel görünür.

Neden Önemli?

  • Mobil Trafiği: Trafiğin %70'i mobil cihazlardan geliyor
  • SEO: Google mobil sayfaları tercih eder
  • UX: Kullanıcı deneyimi çok önemli
  • Maliyet: Tek site, çoklu cihaz desteği

Responsive Yaklaşımları

Yaklaşım Açıklama Kullanım
Mobile-First Mobil'den başla, sonra büyültür Modern (önerilen)
Desktop-First Masaüstü'nden başla, sonra küçültür Eski sitelerde
Adaptive Belirli boyutlar için farklı layout Kompleks siteler
💡 Designer İpucu: Mobile-First yaklaşımı kullanmak, 2024'te en iyi pratiktir!

Viewport Meta Etiketi 🔍

Viewport, tarayıcıya sayfanın genişliğini ve ölçeğini nasıl ayarlayacağını söyler.

Viewport Kodu

Viewport Meta Etiketi

<!-- HTML head'inde mutlaka olmalı --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> /* Açıklaması */ - width=device-width: Sayfanın genişliği cihazın genişliğine eşit - initial-scale=1.0: İlk zoom seviyesi 100%

Viewport aktif ✓

Sayfanız mobilde doğru görüntülenecek

⚠️ Dikkat: Viewport meta etiketi olmadan, mobil cihazlar sayfayı yanlış ölçekte gösterir!

Media Queries 📊

Media Queries, belirli ekran boyutlarında farklı CSS kurallarını uygulamayı sağlar.

Media Query Yapısı

Media Query Örneği

/* Varsayılan (mobile-first) */ body { font-size: 14px; } /* Tablet (600px ve üzeri) */ @media (min-width: 600px) { body { font-size: 16px; } } /* Laptop (1024px ve üzeri) */ @media (min-width: 1024px) { body { font-size: 18px; } }
📱 Mobil: 14px
📱 Tablet: 16px
🖥️ Laptop: 18px

Yaygın Breakpoint'ler

Cihaz Ekran Genişliği Media Query
Mobil (Küçük) < 480px @media (max-width: 480px)
Mobil (Büyük) 480px - 768px @media (min-width: 480px)
Tablet 768px - 1024px @media (min-width: 768px)
Laptop 1024px - 1440px @media (min-width: 1024px)
Büyük Ekran > 1440px @media (min-width: 1440px)

Media Query Türleri

Farklı Media Query Örnekleri

/* Genişlik */ @media (min-width: 768px) { ... } @media (max-width: 480px) { ... } /* Yükseklik */ @media (min-height: 600px) { ... } /* Yönelim */ @media (orientation: portrait) { ... } @media (orientation: landscape) { ... } /* Cihaz tipi */ @media (hover: hover) { ... } @media (pointer: fine) { ... } /* Birleşik */ @media (min-width: 768px) and (max-width: 1024px) { /* Tablet için CSS */ }

Mevcut Ekran Bilgileri:

Genişlik: 0px

Yükseklik: 0px

Yönelim: -

Mobile-First Yaklaşımı 📱

Mobile-first, mobil tasarımdan başlayıp büyük ekranlara doğru ilerlemektir. Bu yaklaşım daha performanslı ve verimlidir.

Mobile-First vs Desktop-First

Mobile-First Desktop-First
Basit başla Karmaşık başla
min-width kullan max-width kullan
Performans iyi Performans düşük
Modern pratik Eski pratik

Mobile-First Örneği

Mobile-First CSS Yazımı

/* Mobil CSS (Temel) */ .container { display: block; width: 100%; padding: 10px; } /* Tablet (600px+) */ @media (min-width: 600px) { .container { padding: 20px; } } /* Laptop (1024px+) */ @media (min-width: 1024px) { .container { display: grid; grid-template-columns: 1fr 1fr 1fr; width: 1200px; margin: 0 auto; } }

✓ Mobile-First en iyi pratiktir!

💡 Designer İpucu: Mobile-first başlayın, sonra büyük ekranlar için min-width media queries ekleyin.

CSS Birimleri 📏

Responsive tasarımda, sabit piksel yerine esnek birimler kullanmak çok önemlidir.

Birim Açıklama Örnek
px Sabit piksel (responsive değil) width: 500px;
% Yüzde (ebeveynine göre) width: 100%;
em Yazı boyutuna göre (yapıştırıcı) padding: 2em;
rem Root yazı boyutuna göre (16px) padding: 2rem;
vw Viewport genişliğinin % 1'i width: 50vw;
vh Viewport yüksekliğinin % 1'i height: 100vh;
max-content İçeriğin doğal genişliği width: max-content;

Birim Örnekleri

px (16px) - Sabit
% (100%) - Ebeveyne göre
em (1.5em) - Yazı boyutuna göre
rem (1rem = 16px) - Root boyutuna göre
💡 Designer İpucu: rem kullanın! Projeniz boyunca tutarlı olur ve değiştirmek kolaydır.

Responsive Resimler 🖼️

Resimler responsif tasarımın önemli parçasıdır. Doğru şekilde optimize etmek lazım.

Responsive Resim Yöntemleri

Resimi Responsive Yapma

/* Yöntem 1: Maksimum genişlik */ img { max-width: 100%; height: auto; display: block; } /* Yöntem 2: Picture etiketi */ <picture> <source media="(min-width: 1024px)" srcset="large.jpg"> <source media="(min-width: 600px)" srcset="medium.jpg"> <img src="small.jpg" alt="Resim"> </picture> /* Yöntem 3: Srcset özniteliği */ <img src="image.jpg" srcset="small.jpg 600w, medium.jpg 1024w, large.jpg 1440w" alt="Responsive Resim">
📷 Responsive Resim

max-width: 100%; height: auto;

Aspect Ratio (En-Boy Oranı)

Aspect Ratio Koruma

/* Modern çözüm */ img { aspect-ratio: 16 / 9; width: 100%; object-fit: cover; } /* Eski çözüm (padding trick) */ .image-container { position: relative; width: 100%; padding-bottom: 56.25%; /* 16:9 */ } .image-container img { position: absolute; top: 0; left: 0; width: 100%; height: 100%; }
16:9 Aspect Ratio

Responsive Tipografi 🔤

Yazı boyutları da ekran boyutuna göre uyum sağlamalıdır.

Responsive Font Sizesi

Font Size Media Query Örneği

/* Mobil (Temel) */ h1 { font-size: 24px; line-height: 1.2; } p { font-size: 14px; line-height: 1.6; } /* Tablet */ @media (min-width: 600px) { h1 { font-size: 32px; } p { font-size: 16px; } } /* Laptop */ @media (min-width: 1024px) { h1 { font-size: 48px; } p { font-size: 18px; } }

Responsive Başlık

Bu metin ekran boyutuna göre otomatik ölçeklendirilir.

Fluid Typography (Akışkan Tipografi)

Clamp Kullanarak Akışkan Font Size

/* Modern CSS (clamp) */ h1 { font-size: clamp(24px, 5vw, 48px); /* Min: 24px, Tercih: 5vw, Max: 48px */ } p { font-size: clamp(14px, 2vw, 20px); } /* Açıklama */ - Viewport genişliğine bağlı olarak akışkan ölçeklenme - En az 24px, en fazla 48px

Akışkan Tipografi

Ekran boyutu değişirse yazı otomatik ölçeklenir

Alıştırmalar 💪

🎯 Alıştırma 1: Responsive Sayfa Oluşturun:
  • Viewport meta etiketi ekleyin
  • Mobil-first yaklaşım kullanın
  • 3 breakpoint (mobil, tablet, laptop) tanımlayın
  • Her breakpoint'te layout değiştirin
🎯 Alıştırma 2: Media Queries Pratikleri:
  • min-width ve max-width kullanın
  • Grid layout'u responsive yapın
  • Flexbox'u responsive yapın
  • Farklı yönelimler için CSS yazın
🎯 Alıştırma 3: CSS Birimlerini Kullanın:
  • rem birimini tüm boyutlar için kullanın
  • % genişlik ve yükseklik kullanın
  • em ve rem arasındaki farkı test edin
  • Clamp() fonksiyonu deneyin
🎯 Alıştırma 4: Responsive Resimler:
  • Resimleri responsive yapın (max-width: 100%)
  • Picture etiketi kullanın
  • Aspect ratio koruyun
  • Farklı çözünürlükler için srcset kullanın
