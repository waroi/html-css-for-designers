HTML Nedir? 🤔

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan işaretleme dilidir. Web tarayıcılarının anlayabileceği bir dildir ve sayfamızın yapısını tanımlar.

HTML sayfamıza içerik ekler, CSS ise bu içeriği güzelleştirir, JavaScript ise sayfayı etkileşimli yapar.

💡 Designer İpucu: HTML sadece yapı, CSS tasarım, JavaScript ise davranış! Birbirlerinin ayrı rolleri vardır.

HTML'in Tarihçesi

  • 1989: Tim Berners-Lee tarafından icat edildi
  • 1999: HTML 4.01 standardı yayınlandı
  • 2014: HTML5 standardı resmi olarak tanımlandı
  • Günümüz: HTML5 aktif olarak geliştirilmeye devam ediyor

Tarayıcı Nasıl Çalışır? 🌐

Bir web sayfasını ziyaret ettiğinizde neler olur?

  1. İstek Gönderme: Tarayıcı sunucuya HTML dosyası istediğini bildirir
  2. Dosya Alma: Sunucu HTML dosyasını tarayıcıya gönderir
  3. Ayrıştırma: Tarayıcı HTML dosyasını satır satır okur (parsing)
  4. DOM Oluşturma: HTML etiketlerinden bir ağaç yapısı (DOM) oluşturur
  5. CSS Yükleme: Stil dosyalarını yükler ve sayfaya uygular
  6. Rendering: Sayfayı ekranda görüntüler
  7. JavaScript Çalıştırma: İnteraktif özellikler için JS kodunu çalıştırır
⚠️ Dikkat: Tarayıcı HTML hatalarını görmezden gelir. Yanlış yazılan etiketler çoğu zaman çalışır ama bunu kötü bir pratiktir!

İlk HTML Sayfasını Oluşturalım! 🚀

Tüm HTML sayfaları belirli bir yapıya sahiptir. Bunu temel yapı bloğu olarak düşünün.

Temel HTML Yapısı

<!DOCTYPE html> <html lang="tr"> <head> <meta charset="UTF-8"> <title>Sayfa Başlığı</title> </head> <body> <h1>Merhaba Dünya!</h1> <p>Bu benim ilk web sayfam.</p> </body> </html>

Merhaba Dünya!

Bu benim ilk web sayfam.

Yapıyı Anlayalım

Etiket Açıklama
<!DOCTYPE html> Belgenin HTML5 olduğunu tarayıcıya söyler
<html> Tüm sayfanın kapsayıcısı (root element)
<head> Tarayıcı için bilgiler (görünmez)
<body> Sayfada görünen tüm içerik
💡 Designer İpucu: Metin editörü olarak VS Code, Atom veya Notepad++ kullanın. Dosyayı .html uzantısıyla kaydedin!

Temel HTML Etiketleri 🏗️

1. Başlık Etiketleri (h1-h6)

Sayfanızda başlıklar oluşturmak için kullanılır. h1 en büyük, h6 en küçüktür.

Başlık Örnekleri

<h1>Büyük Başlık</h1> <h2>Alt Başlık</h2> <h3>Daha Küçük Başlık</h3> <h6>En Küçük Başlık</h6>

Büyük Başlık

Alt Başlık

Daha Küçük Başlık

En Küçük Başlık

2. Paragraf (p)

Metinleri organize etmek için kullanılır.

Paragraf Örneği

<p>Bu bir paragraftır. Web sayfalarında metni paragraflara ayırmak sağdır.</p> <p>Bu başka bir paragraft.</p>

Bu bir paragraftır. Web sayfalarında metni paragraflara ayırmak sağdır.

Bu başka bir paragraft.

3. Bağlantı (a)

Başka sayfalara veya web sitelerine gitmek için kullanılır.

Bağlantı Örneği

<a href="https://www.google.com">Google'ı Ziyaret Et</a> <a href="sayfa2.html">İç Sayfa</a> <a href="#html-nedir">Sayfanın Başına Git</a>
Google'ı Ziyaret Et
İç Sayfa
Sayfanın Başına Git

4. Resim (img)

Sayfaya resim eklemek için kullanılır. Kendi kendini kapatmayan bir etiketdir!

Resim Örneği

<img src="resim.jpg" alt="Açıklama"> <img src="logo.png" alt="Logo" width="100">
📷 Resim
⚠️ Dikkat: alt özniteliği çok önemlidir! Resim yüklenemediğinde bu metin görünür, ayrıca görme engelli kişiler için de gereklidir.

5. Listeler

Öğeleri organize etmek için iki tür liste vardır: sıralı (ol) ve sırasız (ul).

Liste Örnekleri

<!-- Sırasız Liste --> <ul> <li>Elma</li> <li>Armut</li> <li>Muz</li> </ul> <!-- Sıralı Liste --> <ol> <li>Kahvaltı Yap</li> <li>Çalış</li> <li>Uyku</li> </ol>
Sırasız:
  • Elma
  • Armut
  • Muz
Sıralı:
  1. Kahvaltı Yap
  2. Çalış
  3. Uyku

6. Bölme (div) ve Satır İçi (span)

div bloğu, span ise satır içi grup oluşturmak için kullanılır.

Div ve Span Örneği

<div style="background: #f0f0f0; padding: 10px;"> Bu bir div blokudur. </div> <p>Bu paragrafta <span style="color: red;">kırmızı metin</span> vardır.</p>

Alıştırmalar 💪

🎯 Alıştırma 1: Boş bir HTML dosyası oluşturun ve temel yapıyı yazın. Başlık, paragraf ve bağlantı ekleyin.
🎯 Alıştırma 2: Kendiniz hakkında bir sayfa oluşturun:
  • Adınız (h1)
  • Hakkınızda paragraf (p)
  • Hobileriniz (ul liste)
  • Sosyal medya bağlantıları (a etiketleri)
🎯 Alıştırma 3: Sevdiğiniz kitap hakkında bir sayfa oluşturun:
  • Kitap adı (h1)
  • Yazar adı (p)
  • Kitabın kapağı (img)
  • Kitabın ana karakerleri (ul)
  • Çıkış tarihi, sayfa sayısı (p etiketleri)
