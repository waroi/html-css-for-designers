MODÜL 1: HTML Temelleri

Bu modülde HTML'in temellerini öğreneceksiniz. HTML (HyperText Markup Language), web sayfalarının yapısını oluşturan dildir. Tasarımcı olarak, kodun nasıl çalıştığını anlamak sizin yaratıcı çıktılarınızı daha iyi, daha uygulanabilir hale getirecektir.

HTML Nedir? Tarayıcı Nasıl Çalışır?

HTML, "etiketler" (tags) kullanarak web sayfasının iskeletini oluşturur. Tarayıcı bu etiketleri okur ve sayfayı görsel olarak render eder. Örneğin, bir başlık etiketini gördüğünde, tarayıcı bunu büyük bir metin olarak gösterir.

Tarayıcının çalışma süreci:

  1. HTML dosyasını indir
  2. Etiketleri sırayla oku
  3. CSS kurallarını uygula
  4. JavaScript kodunu çalıştır
  5. Görsel olarak sayfayı render et
💡 Designer'lar İçin İpucu: Tarayıcılar etiketleri belirli şekillerde gösterir. Örneğin <h1> her zaman bir başlık gibi görünür. Bu nedenle HTML'de "anlamsal" etiketler kullanmak önemlidir - tasarımınızı doğru yapıda kodlamak, farklı cihazlarda daha iyi çalışmasını sağlar.

HTML'de Etiketler ve Yapısı

Etiketler açılış ve kapanış olmak üzere iki kısımdan oluşur. Bazı etiketler ise kendi kendine kapanır.

<!-- Açılış ve kapanış etiket örneği -->
<h1>Bu bir başlıktır</h1>

<!-- Paragraph etiketinin içinde metin -->
<p>Bu bir paragraftır.</p>

<!-- Kendi kendine kapanan etiket örneği -->
<img src="foto.jpg" />
<br />
<hr />

Temel Etiketler

Başlıklar (h1-h6)

Başlık etiketleri içeriğin hiyerarşisini belirler. <h1> en önemli, <h6> en az önemlisidir.

<h1>Sayfa Başlığı</h1>
<h2>Bölüm Başlığı</h2>
<h3>Alt Bölüm</h3>
<h4>Daha Alt Başlık</h4>

Paragraf (p)

<p> etiketi metni paragraf olarak biçimlendirir. Paragraflar otomatik olarak etrafında boşluk (margin) vardır.

<p>Bu bir paragrafın ilk cümlesidir. Tasarımcılar için HTML öğrenmek şaşırtıcı derecede faydalıdır.</p>
<p>Bu ikinci paragraf. HTML ve CSS bilmek, designerlara daha iyi haberci olmalarını sağlar.</p>

Bağlantılar (a)

<a> etiketi başka sayfalar veya yerler arasında gezinme sağlar. href özniteliği hedefi belirtir.

<a href="https://example.com">Başka Siteye Git</a>
<a href="#modul1-3">Sayfanın içindeki bölüme git</a>

Görseller (img)

<img> web sayfasına resim ekler. src resim dosyasının yolunu, alt ise alternatif metni belirtir.

<img src="portrait.jpg" alt="Bir tasarımcının portresi" />
<img src="/images/icon.png" alt="Uygulama ikonu" />
💡 Designer'lar İçin İpucu: alt özniteliği çok önemlidir! Görme engelli kişiler için ekran okuyucuların resmi tanımlaması için gereklidir. Ayrıca resim yüklenirse bu metin görüntülenir.

Listeler

İki tür liste vardır: sırasız (<ul>) ve sıralı (<ol>). Her liste öğesi <li> ile başlar.

<!-- Sırasız Liste -->
<ul>
  <li>Birinci öğe</li>
  <li>İkinci öğe</li>
  <li>Üçüncü öğe</li>
</ul>

<!-- Sıralı Liste -->
<ol>
  <li>Adım 1</li>
  <li>Adım 2</li>
  <li>Adım 3</li>
</ol>

Konteynerler (div ve span)

<div> bir blok konteyner (tüm satırı kaplar), <span> ise satır içi konteynerdir (sadece gerektiği kadar yer kaplar).

<!-- Div: Blok seviyesinde kapsayıcı -->
<div class="hero-section">
  <h2>Hoş geldiniz</h2>
  <p>Bu başlık bölümüdür.</p>
</div>

<!-- Span: Satır içi kapsayıcı -->
<p>Bu paragrafta <span style="color: red;">kırmızı metin</span> vardır.</p>

HTML Dosya Yapısı

Her HTML dosyası belirli bir yapıyla başlar. Bu yapı tarayıcıya dosyanın ne olduğunu ve nasıl işlenmesi gerektiğini söyler.

<!DOCTYPE html>
<html lang="tr">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width">
    <title>Sayfa Başlığı</title>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <h1>Sayfanın İçeriği</h1>
    <p>Burası kullanıcının göreceği içeriktir.</p>
  </body>
</html>

Her bölümün anlamı:

  • <!DOCTYPE html> - Bu bir HTML5 dosyası olduğunu söyler
  • <html> - Tüm HTML'i kapsayan kök etiket
  • <head> - Sayfanın meta bilgilerini içerir (başlık, stil vb.)
  • <body> - Tarayıcıda gösterilen gerçek içerik

Öznitelikler (Attributes)

Öznitelikler, etiketlere ek bilgi ekler. Her etiket farklı özniteliklere sahip olabilir.

<!-- class özniteliği -->
<div class="container">İçerik</div>

<!-- id özniteliği -->
<section id="bölüm-1">...</section>

<!-- data özniteliği (özel veriler) -->
<button data-action="kaydet">Kaydet</button>

<!-- Birden fazla öznitelik -->
<a href="#" class="link" title="Daha fazla bilgi">Bağlantı</a>

Yaygın öznitelikler:

  • class - CSS ile stillemek için (tekrarlanabilir)
  • id - Benzersiz tanımlayıcı (sadece bir kez kullanılmalı)
  • href - Bağlantıda hedef belirtir
  • src - Dosya kaynağı (resim, script vb.)
  • alt - Alternatif metin (resimler için)
  • data-* - Özel veri depolamak için
🎯 Pratik Alıştırma: Aşağıdaki HTML yapısını oluşturun:
  • Bir başlık "Benim Tasarım Projelerim"
  • Üç paragraf metin
  • Bir görselli 3 başlıklı bir liste (kullanıcı profilleri)
  • Bir "portfolio.css" dosyasına bağlantı

Kaynaklar

  • MDN - HTML Referansı
  • W3Schools - HTML Tutorial
  • WHATWG - HTML Standardı

MODÜL 2: HTML İleri Seviye

Artık temel HTML etiketlerini biliyorsunuz. Bu modülde tablolar, formlar, semantik HTML5 etiketleri ve erişilebilirlik gibi daha gelişmiş konuları öğreneceksiniz. Bu bilgi, web sayfalarını daha profesyonel ve kullanıcı dostu hale getirecektir.

Tablolar ve Formlar

Tablolar

Tablolar veri göstermek için kullanılır. Satır, sütun ve hücrelerden oluşurlar.

<table>
  <thead>
    <tr>
      <th>İsim</th>
      <th>Ünvan</th>
      <th>Şirket</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Ayşe</td>
      <td>Designer</td>
      <td>TechCorp</td>
    </tr>
    <tr>
      <td>Mehmet</td>
      <td>Developer</td>
      <td>WebAgency</td>
    </tr>
  </tbody>
</table>
💡 Designer'lar İçin İpucu: Tablo etiketlerini layout için kullanmayın! Tablolar sadece veri göstermek içindir. Layout için CSS Flexbox veya Grid kullanın.

Formlar

Formlar kullanıcıdan veri toplamak için kullanılır. <form>, <input>, <textarea>, <select> gibi etiketlerden oluşur.

<form action="/submit" method="POST">
  <label for="isim">İsim:</label>
  <input type="text" id="isim" name="isim" />

  <label for="email">E-mail:</label>
  <input type="email" id="email" name="email" />

  <label for="mesaj">Mesaj:</label>
  <textarea id="mesaj" name="mesaj"></textarea>

  <label for="sehir">Şehir:</label>
  <select id="sehir" name="sehir">
    <option value="">Seçin</option>
    <option value="istanbul">İstanbul</option>
    <option value="ankara">Ankara</option>
  </select>

  <button type="submit">Gönder</button>
</form>

Input Tipleri:

  • type="text" - Metin giriş
  • type="email" - E-posta giriş (otomatik validasyon)
  • type="password" - Şifre giriş (gizlenir)
  • type="number" - Sayı giriş
  • type="date" - Tarih seçici
  • type="checkbox" - Seçim kutusu
  • type="radio" - Seçenek düğmesi
  • type="file" - Dosya yükleme

Semantik HTML5 Etiketleri

Semantik etiketler, yapıyı ve anlamı açıklayan etiketlerdir. Bu, arama motorları ve ekran okuyucuları için önemlidir.

<!DOCTYPE html>
<html>
  <body>
    <header>
      <nav>
        <ul>
          <li><a href="/">Ana Sayfa</a></li>
          <li><a href="/about">Hakkında</a></li>
        </ul>
      </nav>
    </header>

    <main>
      <article>
        <h1>Blog Yazısı</h1>
        <p>Bu ana içeriktir...</p>
      </article>

      <aside>
        <h3>Yan Çubuk</h3>
        <ul>
          <li>İlişkili Link 1</li>
          <li>İlişkili Link 2</li>
        </ul>
      </aside>
    </main>

    <footer>
      <p>© 2026 Telif Hakkı Saklıdır</p>
    </footer>
  </body>
</html>

Semantik Etiketler:

  • <header> - Başlık bölümü
  • <nav> - Navigasyon bağlantıları
  • <main> - Ana içerik
  • <article> - Bağımsız içerik (blog yazısı vb.)
  • <section> - Sayfanın mantıksal bölümü
  • <aside> - Yan çubuk veya ilişkili içerik
  • <footer> - Altlık bölümü
💡 Designer'lar İçin İpucu: Semantik etiketler sayfa yapısını daha anlaşılır hale getirir. Arama motorları bu yapıyı okur ve önem derecelerini anlar. Ayrıca ekran okuyucuları kullanan görme engelli kişiler daha iyi bir deneyim yaşar.

Medya Etiketleri

Video Etiketleri

<video width="640" height="480" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  Tarayıcınız video etiketini desteklemiyor.
</video>

Ses Etiketleri

<audio controls>
  <source src="podcast.mp3" type="audio/mpeg">
  Tarayıcınız ses etiketini desteklemiyor.
</audio>

Şekil ve Başlık (figure, figcaption)

<figure>
  <img src="design-process.jpg" alt="Tasarım Süreci" />
  <figcaption>Tasarım Süreci: Araştırma, Taslak, Prototipi</figcaption>
</figure>

İçgün Resim (picture)

<picture>
  <source media="(max-width: 600px)" srcset="mobile-image.jpg">
  <source media="(max-width: 1200px)" srcset="tablet-image.jpg">
  <img src="desktop-image.jpg" alt="Responsive görsel" />
</picture>

Meta Etiketleri ve SEO

Meta etiketleri sayfanın meta bilgilerini tanımlar. Tarayıcılar ve arama motorları bu bilgileri kullanır.

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- SEO Meta Etiketleri -->
  <meta name="description" content="Sayfanız hakkında 155 karakterlik bir açıklama">
  <meta name="keywords" content="html, css, web tasarımı">
  <meta name="author" content="Yazarın İsmi">

  <!-- Open Graph (Sosyal Medya) -->
  <meta property="og:title" content="Sayfa Başlığı">
  <meta property="og:description" content="Açıklama">
  <meta property="og:image" content="https://example.com/image.jpg">
</head>

Erişilebilirlik (Accessibility)

Erişilebilir web, herkes tarafından kullanılabilen web demektir. Bu, görme engelli, işitme engelli ve diğer engellileri de içerir.

ARIA Roles

<!-- Başlık Rolü -->
<div role="banner">...</div>

<!-- Navigasyon Rolü -->
<div role="navigation">...</div>

<!-- Uyarı Dialogu -->
<div role="alert">Bu önemli bir uyarıdır!</div>

<!-- Tab Listesi -->
<ul role="tablist">
  <li role="tab">Tab 1</li>
  <li role="tab">Tab 2</li>
</ul>

Alt Text

<!-- Kötü: Tanımlayıcı olmayan alt metin -->
<img src="photo.jpg" alt="resim" />

<!-- İyi: Tanımlayıcı alt metin -->
<img src="design-team.jpg" alt="Tasarım ekibi ofiste çalışıyor" />

Label ve Form Erişilebilirliği

<!-- Her input'un bir label'ı olmalı -->
<label for="search-box">Ara:</label>
<input id="search-box" type="search" placeholder="Arama teriminizi girin" />

<!-- Checkbox'un label'ı -->
<input type="checkbox" id="agree">
<label for="agree">Şartları kabul ediyorum</label>
💡 Designer'lar İçin İpucu: Erişilebilirlik, sadece etik değil aynı zamanda iş açısından da mantıklıdır. Daha geniş bir kitle sayfanızı kullanabilecek, SEO daha iyi olacak ve yasal sorunları önleyebileceksiniz.
🎯 Pratik Alıştırma: Semantik yapılı, erişilebilir bir blog post sayfası oluşturun:
  • Başlık, navigasyon
  • Ana makale (başlık, paragraflar, resimler figcaption'yla)
  • Yan çubuk (ilişkili yazılar)
  • Yorum formu (label'larla)
  • Altlık (meta bilgiler)

Kaynaklar

  • MDN - Erişilebilirlik

MODÜL 3: CSS Temelleri

CSS (Cascading Style Sheets), HTML elemanlarının nasıl görüneceğini kontrol eder. Tasarımcı olarak, CSS'de ustalaşmak sizin tasarımlarınızı web'de tam olarak gerçekleştirmenizi sağlar.

CSS Nedir ve Nasıl Bağlanır?

CSS, "kurallar" kümesidir. Her kural hangi elemanları seçeceğini (seçici) ve onlara ne stil vereceğini (özellikler) belirtir.

/* CSS Kuralının Yapısı */
selector {
  property: value;
  another-property: value;
}

CSS Bağlama Yöntemleri

1. External CSS (En İyi Yöntem)

<!-- HTML dosyasında -->
<link rel="stylesheet" href="style.css">

/* style.css dosyasında */
body {
  font-family: Arial, sans-serif;
}

2. Internal CSS

<style>
  body {
    font-family: Arial, sans-serif;
  }
</style>

3. Inline CSS (Kaçınılmalı)

<p style="color: red; font-size: 16px;">Kırmızı metin</p>

Seçiciler (Selectors)

Element Seçici

p {
  color: blue;
}

Class Seçici

/* CSS */
.highlight {
  background-color: yellow;
}

/* HTML */
<p class="highlight">Bu vurgulanmış metindir</p>

ID Seçici

/* CSS */
#main-title {
  font-size: 32px;
}

/* HTML */
<h1 id="main-title">Sayfa Başlığı</h1>

Grup Seçici

h1, h2, h3 {
  font-weight: bold;
  color: navy;
}

Soyundan Gelen Seçici (Descendant)

div p {
  color: green;
}

Çocuk Seçici (Child)

ul > li {
  list-style: none;
}

Bitişik Kardeş Seçici

h2 + p {
  font-weight: bold;
}
💡 Designer'lar İçin İpucu: Seçici özgünlüğü önemlidir. ID'ler en spesifik, ardından class'lar, sonra element seçicileri gelir. Genellikle class'ları kullanmak en iyi uygulamadır.

Renk Sistemleri

Hex Renkler

body {
  background-color: #ffffff; /* Beyaz */
  color: #000000;           /* Siyah */
  border-color: #6366f1;    /* İndigo */
}

RGB/RGBA Renkler

.box {
  background-color: rgb(99, 102, 241);      /* İndigo */
  border-color: rgba(99, 102, 241, 0.5);   /* İndigo %50 saydamlık */
}

HSL Renkler

.element {
  background-color: hsl(226, 71%, 67%);    /* İndigo */
  border-color: hsl(226, 71%, 67%, 0.5);  /* %50 saydamlık */
}

HSL Açıklaması:

  • Hue (Renk): 0-360 derece (0=Kırmızı, 120=Yeşil, 240=Mavi)
  • Saturation (Doygunluk): 0-100% (0=Gri, 100=Dolu renk)
  • Lightness (Parlaklık): 0-100% (0=Siyah, 50=Orta, 100=Beyaz)

Tipografi

Font Family

body {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
}

.serif-text {
  font-family: Georgia, 'Times New Roman', serif;
}

.monospace {
  font-family: 'Courier New', Courier, monospace;
}

Font Size, Weight, Style

h1 {
  font-size: 48px;
  font-weight: bold;     /* 400, 700 vb. de yazılabilir */
}

em {
  font-style: italic;
}

.light-text {
  font-weight: 300;
}

.extra-bold {
  font-weight: 900;
}

Line Height ve Letter Spacing

p {
  line-height: 1.6;             /* Satır yüksekliği */
  letter-spacing: 0.05em;       /* Harfler arası boşluk */
}

h2 {
  line-height: 1.2;
  letter-spacing: -0.01em;      /* Yoğun harfler */
}

Box Model

CSS'de her eleman bir "box" (kutu) olarak davranır. Bu kutunun dört katmanı vardır: content, padding, border, margin.

.box {
  width: 300px;
  height: 200px;
  padding: 20px;              /* İçerik etrafı boşluk */
  border: 3px solid #333;     /* Kenarlık */
  margin: 30px;               /* Dış boşluk */
}

/* Kısa yazım */
.box2 {
  padding: 10px 20px 10px 20px;  /* Top, Right, Bottom, Left */
  margin: 15px 0;                /* Top/Bottom, Left/Right */
}

Box Model Görseli:

┌─────────────────────────────────┐
│         Margin (Dış)            │
│  ┌───────────────────────────┐  │
│  │  Border (Kenarlık)        │  │
│  │ ┌─────────────────────┐   │  │
│  │ │ Padding (İçerik)    │   │  │
│  │ │ ┌─────────────────┐ │   │  │
│  │ │ │  Content        │ │   │  │
│  │ │ └─────────────────┘ │   │  │
│  │ └─────────────────────┘   │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘

Display Türleri

Block Display

div, p, h1 {
  display: block;            /* Tüm satırı kaplar */
}

.block-element {
  width: 100%;               /* Tam genişlik */
  margin-bottom: 10px;
}

Inline Display

span, a, strong {
  display: inline;           /* Sadece gereken yer */
}

.inline-element {
  margin: 0 10px;            /* Top/Bottom margin çalışmaz */
}

Inline-Block Display

.inline-block-item {
  display: inline-block;     /* Block gibi, ama satır içi */
  width: 150px;
  margin: 10px;
}

None Display

.hidden {
  display: none;             /* Sayfada yer almaz */
}

/* Görünmez ama yer alır (visibility) */
.invisible {
  visibility: hidden;
}
🎯 Pratik Alıştırma: Basit bir "Kartlar" (cards) tasarımı yapın:
  • Her kartın başlık, resim ve açıklama olsun
  • Renkli border ve box-shadow kullanın
  • Padding ve margin'i düzgün ayarlayın
  • 3 kartı yan yana gösterin

Kaynaklar

  • W3Schools - CSS Öğrenin

MODÜL 4: CSS Orta Seviye

Artık CSS temellerini biliyorsunuz. Bu modülde, modern web tasarımının temel taşları olan Flexbox, Grid, pozisyonlama ve animasyonları öğreneceksiniz.

Flexbox - Esnek Kutu Düzeni

Flexbox, ögeleri satır veya sütun halinde düzenlemeyi kolaylaştırır. Modern web tasarımı için başlıca yöntemlerden biridir.

Flexbox'u Başlatma

.container {
  display: flex;             /* Flex konteyner oluştur */
  flex-direction: row;       /* Yön: row, column, row-reverse, column-reverse */
  justify-content: center;   /* Yatay hizalama */
  align-items: center;       /* Dikey hizalama */
}

Justify-Content Değerleri

.container {
  justify-content: flex-start;    /* Başta */
  justify-content: center;        /* Ortada */
  justify-content: flex-end;      /* Sonunda */
  justify-content: space-between; /* Eşit aralıklı */
  justify-content: space-around;  /* Etrafında eşit boşluk */
  justify-content: space-evenly;  /* Tam eşit boşluk */
}

Align-Items Değerleri

.container {
  align-items: flex-start;   /* Üstte */
  align-items: center;       /* Ortada */
  align-items: flex-end;     /* Altta */
  align-items: stretch;      /* Uzat (varsayılan) */
  align-items: baseline;     /* Metin tabanına hizala */
}

Flex Ögeleri

.container {
  display: flex;
  gap: 20px;                 /* Öğeler arasında boşluk */
}

.item {
  flex: 1;                   /* Eşit boyut */
  min-width: 100px;          /* Minimum genişlik */
}

.item-2 {
  flex: 2;                   /* 2 katı daha geniş */
}

.item-fixed {
  flex: 0 0 200px;           /* Sabit 200px */
}

Wrap - Ögeleri Kaydırma

.container {
  display: flex;
  flex-wrap: nowrap;         /* Varsayılan, kaydırma yok */
  flex-wrap: wrap;           /* Kaydır */
  flex-wrap: wrap-reverse;   /* Ters kaydır */
}
💡 Designer'lar İçin İpucu: Flexbox, navigasyon menüleri, kartlar ve iki sütunlu düzenler için mükemmeldir. Basit ve sezgisel bir çözüm sunarak CSS'i tasarımcılar için erişilebilir hale getirir.

CSS Grid - Izgara Düzeni

CSS Grid, iki boyutlu (satırlar ve sütunlar) düzenler için idealdir. Daha karmaşık sayfa yapıları oluşturmada çok güçlüdür.

Grid'i Başlatma

.grid-container {
  display: grid;
  grid-template-columns: 200px 1fr 200px;  /* 3 sütun */
  grid-template-rows: auto 1fr auto;       /* 3 satır */
  gap: 20px;                               /* Sütunlar ve satırlar arası boşluk */
}

Otomatik Sütunlar

  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}

Grid Elemanlarını Konumlandırma

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 100px 200px 100px;
}

  grid-row: 1 / 2;
}

  grid-column: 1 / 4;      /* 3 sütunun tamamı */
  grid-row: 2 / 3;
}

.item-3 {

  grid-row: 3 / 4;
}

Grid Template Areas

.layout {

  grid-template-areas:
    "header header header"

CSS Değişkenleri (Custom Properties)

CSS değişkenleri, renk, boyut gibi değerleri merkezi olarak yönetmeyi sağlar.

  --border-radius: 8px;

  --dark-bg: #1f2937;

}

MODÜL 5: Responsive Design

Responsive tasarım, web sitelerinin tüm cihazlarda (mobil, tablet, masaüstü) güzel görünmesini sağlar. Modern web tasarımında bu, artık isteğe bağlı değil gereklidir.

Responsive Design Nedir?

Responsive tasarım, sayfanın ekran boyutuna göre uyum sağlamasıdır. Flex, Grid ve media queries kullanarak bunu başarırız.

💡 Designer'lar İçin İpucu: "Mobile First" yaklaşımını kullanın. Önce mobil tasarımı yapın, sonra daha geniş ekranlar için iyileştirme yapın. Bu, temiz ve basit kod yazmanıza yardımcı olur.

Media Queries

Media queries, farklı ekran boyutlarına göre farklı CSS kuralları uygulamanızı sağlar.

/* Tablet ve daha geniş */
@media (min-width: 768px) {
  .container {
    display: grid;
    grid-template-columns: 1fr 1fr;
  }
}

/* Masaüstü */
@media (min-width: 1024px) {
  .container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}

/* Sadece mobil */
@media (max-width: 767px) {
  h1 {
    font-size: 24px;
  }
}

/* Portait ve landscape */
@media (orientation: landscape) {
  body {
    margin: 10px;
  }
}

Yaygın Breakpoint'ler:

  • Mobil: 320px - 480px
  • Tablet: 481px - 768px
  • Küçük Masaüstü: 769px - 1024px
  • Masaüstü: 1025px ve üstü

Görüntü Alanı ve Birimleri

Viewport Meta Etiketi

<meta name="viewport"
       content="width=device-width, initial-scale=1.0">

Relatif Birimler

body {
  font-size: 16px;           /* Temel boyut */
}

h1 {
  font-size: 2em;            /* 32px (16 * 2) */
}

h2 {
  font-size: 1.5rem;         /* 24px (her zaman 16'ya göre) */
}

p {
  font-size: 1rem;           /* 16px */
  line-height: 1.5;          /* 24px */
}

/* Viewport Birimleri */
.full-screen {
  width: 100vw;              /* Viewport genişliği */
  height: 100vh;             /* Viewport yüksekliği */
}

.responsive-box {
  width: 80vw;               /* Viewport'un %80'i */
  height: 50vh;              /* Viewport'un %50'si */
}

Birim Karşılaştırması:

  • px - Sabit piksel
  • em - Ana elemanın yazı boyutuna göre
  • rem - Root (html) yazı boyutuna göre
  • % - Ana elemanın yüzdesine göre
  • vw/vh - Viewport'un yüzdesine göre

  }

  }

MODÜL 6: İleri Seviye Konular

Web tasarımında ustalaşmak için bilmeniz gereken ileri konular: CSS metodolojileri, dark mode, modern CSS özellikleri ve daha fazlası.

CSS Metodolojileri - BEM

BEM (Block Element Modifier), CSS sınıflandırma sisteminin bir standardıdır. Büyük projelerde kodun okunabilirliğini ve yeniden kullanılabilirliğini arttırır.

/* BEM Yapısı: Block__Element--Modifier */

/* Block: Bağımsız bileşen */
.button { }

/* Element: Block'ın parçası */
.button__icon { }
.button__text { }

/* Modifier: Değişken durum */
.button--primary { }
.button--large { }
.button__icon--small { }

/* HTML Örneği */
<button class="button button--primary">
  <span class="button__icon button__icon--small">🔍</span>
  <span class="button__text">Ara</span>
</button>

Dark Mode Implementasyonu

Dark mode, modern tasarımda standart hale geldi. CSS değişkenleriyle kolayca uygulanabilir.

:root {
  /* Light Mode (varsayılan) */
  --bg-primary: #ffffff;
  --text-primary: #111827;
  --bg-secondary: #f3f4f6;
}

@media (prefers-color-scheme: dark) {
  :root {
    /* Dark Mode */
    --bg-primary: #1f2937;
    --text-primary: #f3f4f6;
    --bg-secondary: #111827;
  }
}

/* Veya manuel toggle ile */
body.dark-mode {
  --bg-primary: #1f2937;
  --text-primary: #f3f4f6;
  --bg-secondary: #111827;
}

body {
  background: var(--bg-primary);
  color: var(--text-primary);
  transition: background 0.3s, color 0.3s;
}

JavaScript ile Dark Mode Toggle

const themeToggle = document.getElementById('themeToggle');

themeToggle.addEventListener('click', () => {
  document.body.classList.toggle('dark-mode');
  safeStorage('set', 'theme', document.body.classList.contains('dark-mode') ? 'dark' : 'light'
  );
});

/* Sayfa yüklendiğinde tema yükleme */
if (safeStorage('get', 'theme') === 'dark') {
  document.body.classList.add('dark-mode');
}

clamp() - Dinamik Boyutlandırma

.responsive-text {

}

}

  container-type: inline-size;
}

.video-container {
  aspect-ratio: 16 / 9;
  /* Genişlik nasıl ayarlanırsa, yükseklik otomatik */

Erişilebilirlik Derinlemesine

Erişilebilir tasarım, herkes tarafından kullanılabilen tasarımıdır. CSS ile yapabilecekleriniz:

/* Keyboard navigasyonu görünür kıl */
:focus-visible {
  outline: 3px solid #6366f1;
  outline-offset: 2px;
}

/* Hareket duyarlılığı */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}

/* Kontrast iyileştirme */
@media (prefers-contrast: more) {
  body {
    --text-color: #000000;
    --bg-color: #ffffff;
  }
}

/* Skip Link */
.skip-link {
  position: absolute;
  top: -40px;
  left: 0;
  background: #000;
  color: white;
  padding: 8px;
  text-decoration: none;
}

.skip-link:focus {
  top: 0;
}
